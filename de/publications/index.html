KubeOps — dotnet-operator-sdk

KubeOps ist ein Kubernetes Operator SDK für .NET, stark an kubebuilder angelehnt. Es ermöglicht die Entwicklung produktionsreifer Kubernetes-Operatoren mit C# und ASP.NET Core — und schlägt damit eine Brücke zwischen dem .NET-Ökosystem und Cloud-Native-Infrastruktur.

WP-Mail-Validator

An anti-spam plugin for Wordpress CMS. It provides mail-address verification in 5 ways:

  • syntax of mail-addresses
  • mailserver host
  • mx-record of mailserver
  • user-defined blacklist
  • trashmail services
